All Beef No Bull

HTML \ CSS \ JS \ JQUERY

Demo Github

What

All Beef No Bull is a website developed for a Toronto based small business that specializes in gourmet dog treats. The design is minimal, but effective and memorable.

Why

Starting a new business online can be daunting, especially with high monthly costs from most ecommerce platforms out there. This project was developed to overcome that obstacle.

How

Stack: HTML, CSS, JavaScript, JQUERY

Other tools used: VS Code, Fimga, PhotoShop, Netlify, Square

The project structure is simple - static HTML, CSS and JS files - all served up through Netlify.

This project gives a new business owner the best of both worlds. Instead of paying $30 USD for a Shopify account, we used a free Square account to create a store that we link to throughout the site. The rest of the site (home, our story, testimonials and contact page are all static pages served up through Netlify.

We kept things even more simple, using jQuery to help out with the menu slide out functionality.
