Giphy Genie

HTML \ CSS \ JS \ GIPHY API

Demo Github

What

Giphy Genie is a random GIF generator using the Giphy API. Simply type in a keyword into the input field and hit enter to see a random GIF populate the page based on your keyword.

Why

This project was originally done through a course offered from SuperHi, but adapted with my own flare. The purpose of this project was to get a better understanding of how data is passed through an API and get more comfortable with using Postman.

How

Stack: HTML, CSS, JavaScript, jQuery, Giphy API

Other tools used: VS Code, Postman, Fimga, Netlify

The project structure is as easy as it gets - static HTML, CSS and JS files - all served up through Netlify.

In terms of the code setup, we're using vanilla JavaScript to handle functions such as:

  • Randomizing the search results outcome
  • Creating and injecting the video element to the page
  • Everything to do with handling the search function in general
